# 24K COMPANY — Plataforma SaaS

## Como Abrir

1. Abra a pasta `24K COMPANY` nos seus Documentos
2. Dê duplo clique em `index.html`
3. Use os botões de **Acesso Rápido (Demo)** para navegar:
   - 👤 **Cliente** → Dashboard, Materiais, Campanhas, Feedback
   - 👥 **Colaborador** → Dashboard Operacional, Upload, Agente IA
   - 💼 **Gestor** → Painel Executivo, Carteira Completa

## Arquivos do App

| Arquivo | Descrição |
|---|---|
| `index.html` | Login / Cadastro |
| `onboarding.html` | Onboarding IA com 10 perguntas |
| `client-dashboard.html` | Dashboard do Cliente |
| `materials.html` | Aprovação de Materiais |
| `campaigns.html` | Métricas de Campanhas |
| `feedback.html` | NPS + Chat de Suporte |
| `credentials.html` | Armazenamento de Senhas |
| `team-dashboard.html` | Dashboard Operacional da Equipe |
| `upload-materials.html` | Upload de Materiais |
| `agent.html` | Agente IA de Campanhas |
| `executive-dashboard.html` | Painel Executivo |
| `all-clients.html` | Carteira Completa |

## Requisitos

- Nenhum! Abre diretamente no Chrome, Edge ou Firefox
- Não precisa de Node.js, servidor ou instalação
- Dados salvos localmente (localStorage)
